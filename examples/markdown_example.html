# Euler Problem 30 ## Problem This problem is described in project Euler, as [problem 30](https://projecteuler.net/problem=30) copied here verbatim: > Surprisingly there are only three numbers that can be written as the sum of fourth powers of their digits: > 1634 = 14 + 64 + 34 + 44 > 8208 = 84 + 24 + 04 + 84 > 9474 = 9 4 + 4 4 + 7 4 + 4 4 > As 14 is not a sum it is not included. > The sum of these numbers is 1634 + 8208 + 9474 = 19316 > Find the sum of all the numbers that can be written as the sum of fifth powers of their digits. ## Solution We set the power to raise to be 5, as per the problem's definition. Brute force search should be enough for this problem, but we need to set the upper bound for that. How do we find it? Well, 9^5 = `Math.pow(9,5)` , which is a 5-digit number; i.e. the solution will have at least 5 numbers. This brings us to 5 \* 95 = `5 * Math.pow(9,5)` , which is a 6 digit number. Hence, we come to 6 \* 95 = `6 * Math.pow(9,6)`, and rounding up a bit will leave us at 355000. ``` var power = 5 var upperBound = 355000; ``` We now pre-compute the 5th power of all the digits, so we can reuse that quickly in our solution: ``` {"id":"digits"} var digitsPowered = []; for (i = 0; i < 10; i++) digitsPowered.push(Math.pow(i,power)) ``` And now we're ready to solve, essentially a brute force search: ``` var sum = function(arr) { return arr.reduce(function(pv,cv) { return pv+cv},0); }; var sumPowerDigits = function(num) { return sum(num.toString() .split("") .map(function(d) { return digitsPowered[parseInt(d)]; })); }; var isSumOfDigitsPowered = function(num) { return num == sumPowerDigits(num); } var validValues = []; for (i = 10; i < upperBound; i++) if (isSumOfDigitsPowered(i)) validValues.push(i); var result = sum(validValues); ``` And this results in : `result`
